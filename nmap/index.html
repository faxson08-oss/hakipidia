What is Nmap?

Nmap (Network Mapper) is a free and open-source tool for network discovery and security auditing.

Basic Scan

This scans the target for open TCP ports:

nmap example.com

SYN Stealth Scan

Also known as a half-open scan, itâ€™s less likely to be detected:

nmap -sS example.com

Service Version Detection

Detect service versions running on open ports:

nmap -sV example.com

Combine Stealth and Version Scan

This combines both scans in one command:

nmap -sS -sV example.com

Scan Multiple IPs or Subnets

You can scan multiple targets:

nmap 192.168.1.1 192.168.1.2
nmap 192.168.1.0/24

Save Scan Results

Save output to a file:

nmap -oN scan_results.txt example.com
